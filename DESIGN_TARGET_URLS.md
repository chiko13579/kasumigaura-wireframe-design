# Design Target URLs

このファイルは、デザイン調整が必要な主要ページの一覧。
`高校_デザイン版` は現時点で公開リモート未設定のため、まずはローカル確認用URLとして記載。

ローカルサーバーをこのフォルダで起動した場合のベースURL:

```text
http://127.0.0.1:5177/
```

## 公開後のURLについて

GitHub Pagesなどで公開する場合は、上記URLのベース部分を公開URLに置き換える。

例:

```text
https://example.github.io/kasumigaura-design/admission.html
```
